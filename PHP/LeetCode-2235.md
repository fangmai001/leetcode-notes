# LeetCode #2235 Add Two Integers

```php
class Solution {

    /**
     * @param Integer $num1
     * @param Integer $num2
     * @return Integer
     */
    function sum($num1, $num2) {
        return $num1 + $num2;
    }
}

$solution = new Solution();
print_r($solution->sum(12, 5));
print_r("\n");
print_r($solution->sum(-10, 4));
print_r("\n");
```
